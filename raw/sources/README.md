# Published sources mapping

The existing `/content/posts` tree remains the publication layer.

Inside the LLM Wiki model, published posts are treated as source artifacts for higher-level wiki topics.

Interpretation rules:
- a post is not automatically the canonical wiki page
- high-value posts should feed a stable `/wiki/topics` or `/wiki/concepts` page
- `content/posts` remains the reader-facing publishing layer

Primary source tree:
- `../../content/posts/`
