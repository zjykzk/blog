# Journals as raw inputs

The existing `/journals` directory remains the active daily-note stream.

This folder defines how to interpret it inside the LLM Wiki model:
- journals are time-based raw inputs
- stable takeaways should be extracted into `/wiki`
- original daily notes remain the source trail

Primary examples:
- `../../journals/2026_04_01.md`
- `../../journals/2026_04_04.md`
