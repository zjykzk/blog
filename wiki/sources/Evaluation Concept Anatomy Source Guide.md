# Evaluation Concept Anatomy Source Guide

This page preserves the knowledge extracted from the `ljg-learn` concept-anatomy note on [[wiki/concepts/Evaluation]].

## Source Claim

The source defines evaluation as placing an object inside a standard in order to judge its value, quality, meaning, or position.

It explicitly warns against reducing evaluation to liking, scoring, or criticism. Evaluation aligns emotion, standard, evidence, and situation into a judgment.

## Useful Distinctions

- Evaluation versus preference: preference can be immediate liking; evaluation requires a scale.
- Evaluation versus scoring: scoring is one possible representation, not the whole judgment.
- Judgment metaphor versus tuning metaphor: judgment imagines a verdict, while tuning imagines hearing misalignment and helping the object approach its own more accurate form. ^[inferred]
