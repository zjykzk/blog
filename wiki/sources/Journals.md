# Journals Source Guide

`journals/` 是当前 vault 的时间流输入层。它们不应直接承担稳定知识页职责，但应该持续为 `wiki/` 提供可回溯的来源。

## Current examples

- [[journals/2026_04_01]]
- [[journals/2026_04_04]]

## Extraction rule

- 时间驱动内容先留在 `journals/`
- 可复用判断再抽到 `wiki/concepts/`、`wiki/topics/`、`wiki/maps/`
- wiki 页面应保留回链，指回原 journal note
