# 推理证明与未知问答 Source Guide

> Source: conversation:2026-05-19

## Capture Policy

这页保留一段围绕“推理、证明、未知”的中文教学追问。它不是替代 [[wiki/sources/推理概念解剖 Source Guide]]，而是作为 companion source guide，保存后续问答中形成的解释层、对比关系和最小图式。

## What It Covers

这段问答把推理从“从已知到未知”进一步拆开：未知不是任意空白，而是被前提、概念、关系和证据约束出来的判断空位。它也区分了推理和证明：推理是更大的活动，证明是其中更严格、更公开、更可复查的一种形态。

## Preserved Content

### 原始问题链

```text
1. 推理和证明有什么关系？
2. 推理中的未知指的是什么？
```

这两个问题形成一条连续教学线：先区分“推理”和“证明”，再回头拆解推理定义中“未知”的含义。

### 推理和证明的关系

推理是更大的活动，证明是推理的一种更严格形态。

推理从前提、迹象、证据出发，经过概念命名、关系规则、推演、排除、比较，走向结论，并可能给出置信度。推理可以是演绎、归纳、类比、溯因，也可以是日常判断。

证明则把某个结论放进更严格的可检查链条里，让别人能沿着前提、规则和步骤复查，确认“这个结论为什么必须成立”或“在这些条件下足够成立”。

最短关系式是：

```text
证明 = 被公开化、规则化、可回查的推理
```

这个关系包含三个差异：

1. 推理可以探索，证明要求交付。推理可以提出可能解释；证明要把其中一个结论固定下来。
2. 推理可以有置信度，证明追求必然性或充分正当性。日常推理常说“更可能”；数学证明要说“必然”；科学证明和法律证明更多是在各自规范下说明“证据足够支持”。^[inferred]
3. 推理重在从已知到未知，证明重在让这条路可复查。证明把推理链条显式展开，并接受他人检查。

### 推理中的“未知”

推理中的“未知”不是神秘对象，而是当前还没有直接给出的判断。

它至少有四种形态：

1. **缺失的结论**：已知前提有了，但结论还没有明说。比如地湿、窗开、阳台有风，可以推出“可能下过雨”。
2. **缺失的原因**：现象看到了，但背后的原因还不知道。比如服务变慢，可能是数据库、网络、锁竞争或流量突增。
3. **缺失的关系**：事实都在，但它们之间如何连接还不清楚。比如用户流失、功能复杂、客服压力上升，可能只是并列现象，也可能是同一结构生成的结果。
4. **缺失的位置或变量**：在一个结构里有空位等待补全。[[wiki/concepts/Reasoning as Structure Mapping]] 中的结构映射图表达的就是这种未知。

```text
a ---- b        x ---- y
|      |   →    |      |
c ---- d        z ---- ?
```

这里的 `?` 就是未知。它不是完全随机的空白，而是被已有结构约束出来的空位。

### 受约束的未知

推理中的未知可以压缩成一句话：

```text
未知 = 尚未直接给出、但可以被前提、概念、关系和证据约束出来的判断空位
```

这一区分很重要：

- 没有约束的未知，容易变成想象、脑补或任意假设。
- 有前提、关系规则和证据回查的未知，才进入推理。
- 能被公开展开并复查的推理链，才接近证明。

### 三层关系图

```text
已知材料
  = 事实 / 前提 / 迹象 / 证据
        |
        v
推理
  = 用概念命名前提，用关系规则推进，生成受约束的未知判断
        |
        v
证明
  = 把推理链条公开化、规则化、可回查，使结论接受复查
```

### 教学用最小区分

| 问题 | 核心回答 |
|---|---|
| 推理是什么？ | 从已有材料出发，沿关系纹路走向尚未直接给出的判断。 |
| 未知是什么？ | 被前提、概念、关系和证据约束出来的判断空位。 |
| 证明是什么？ | 被公开化、规则化、可回查的推理。 |
| 猜想和推理的边界在哪里？ | 是否有前提、关系规则和证据回查。 |
| 推理和证明的边界在哪里？ | 推理是否被展开成可复查的规范链条。 |

## Integration Decisions

这页应作为 [[wiki/sources/推理概念解剖 Source Guide]] 的教学型补充。原 source guide 保存“推理”的概念解剖；本页保存后续问答中出现的两个关键补丁：证明是推理的严格化形态，未知是受约束的判断空位。

它也补强 [[wiki/concepts/Reasoning as Structure Mapping]]。结构映射页中 `?` 的含义可以用本页的“受约束未知”来解释：`?` 不是任意答案，而是由源结构、目标结构、对应关系和约束检验共同限定出来的空位。

本页不新建“证明”或“未知”概念页，因为当前材料还不足以支撑稳定概念页。后续如果有数学证明、科学证明、法律证明或不确定性来源材料，可以再把“证明”提升为独立概念或综合页。^[inferred]
