# Analysis Concept Anatomy Source Guide

This page preserves the knowledge extracted from the `ljg-learn` concept-anatomy note on [[wiki/concepts/Analysis]].

## Source Claim

The source defines analysis as separating a whole into parts, observing relations among those parts, and understanding how the whole is constituted.

It explicitly warns against equating analysis with merely breaking something apart. A good analysis must be able to return the parts to structure.

## Useful Distinctions

- Analysis versus synthesis: analysis opens, synthesis recomposes.
- Mechanical metaphor versus flow metaphor: the mechanical metaphor sees parts and interfaces; the flow metaphor also preserves direction, resistance, confluence, and time.
- Useful decomposition versus avoidance: analysis can clarify action, but over-analysis can become a retreat from action.
