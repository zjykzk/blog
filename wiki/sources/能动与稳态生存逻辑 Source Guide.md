# 能动与稳态生存逻辑 Source Guide

> Source: 《能动：稳态生存的观念陷阱》，用户粘贴的中文文章。

## What It Covers

这篇文章把“能动”放在高波动世界与旧观念滞后之间理解：现代社会已经从低流动、低波动、地方性稳定生活，转向职业更替、城市迁移、机会重组和风险上升的环境；但很多人的观念仍停留在稳态生存逻辑中。

文章的核心不是批评某个阶层，而是指出一套策略性心智模型：资源匮乏、强从众、简单模型曾经适合稳态生存，却会在高波动世界中抑制 [[wiki/concepts/Human Agency]]、[[wiki/topics/Wealth Creation]] 和进入 [[wiki/concepts/Multiplicative World]] 的能力。

## Key Points

- “人生说明书”的思想游戏强调：如果来世随机投胎，最值得传递的不是具体知识和财富，而是能够摆脱平庸观念、迁移到大城市、学习可迁移技能、争取不对称优势和积累复利的世界观。
- 稳态生存逻辑的背景是长期低流动生活：一个地方、一项技能、一份工作、一群熟人可以维持一辈子。
- 文化滞后说明，物质条件改变后，观念不会同步升级；旧叙事仍会限制人在新环境中的行动范围。
- 稳态生存逻辑有三大基因：资源匮乏、强从众、用简单模型看世界。
- 资源匮乏会让人生存优先于发展，并把原本不得已的生存策略美化成价值、道德或审美。
- “次级奖励”解释了人为什么会通过学习喜欢原本并不喜欢的东西：匮乏时代形成的节俭、凑合、廉价果腹或残缺美学，后来会被叙事包装成文化传统。
- 人情、孝道和零和思维都可以从匮乏结构中推演出来：当服务、养老和资源分配缺乏制度保障，人际债务、道德约束和防御性竞争就会变得合理。
- 风险厌恶是匮乏最深的心理后果之一。生存本金太少时，不确定性不是朋友，而是威胁。
- “勤俭”“安土重迁”“枪打出头鸟”“万事俱备再行动”等观念都包含降低风险、避免暴露和减少失败的稳态策略。
- 现代商业中的 MVP 思路要求先用最小版本进入反馈循环；完美计划主义常常不是严谨，而是害怕失败。
- 强从众来自弱规则：当法律、制度和明确边界不能提供足够安全感，人就会依赖人际关系、权威和多数行为来降低不确定性。
- 弱规则社会容易奖励“听话”。但听话不是能力，而是把判断外包给他人的姿态；它会把人推向平庸。
- 态度文化把服从表现成业绩：结果是否有效反而不重要，忙碌、苦劳、姿势和地位信号成为替代性指标。
- 面子文化让人优先考虑别人如何评价，而不是行动是否带来真实收益。买房、豪车、彩礼等都可能成为从众压力下的地位表达。
- 简单模型会把外部世界看成一个整体，无法区分不同责任关系、交易边界和因果链条。
- “努力”常被用成线性模型：只要我足够努力，世界就应该回报我。但现实回报取决于方向、领域、杠杆、时机、交易结构和创造的价值。
- 指标主义把成绩、学历、岗位等指标当成世界欠自己的凭证，却忽略自己究竟能为世界提供什么。
- 稳态品质具有局部理性：听话、节俭、努力、随大流、不出头、孝顺，会让人省心、肯干、不闯祸。
- 这些品质的全局风险是让人适合被使用，而不一定适合兴旺发达；学校和组织常训练合格劳动者与守法居民，而不是训练能创造大波动的人。^[inferred]
- 真正爱孩子的家长不应只期待孩子听话、努力和抗拒波动，而应希望孩子拥有更复杂的思维模型、科学工具和创造波动的勇气。
- 文章最终把问题收束到能动者：人应成为调用工具的人，而不是别人的工具。

## Integration Decisions

- [[wiki/concepts/Human Agency]] 值得建立概念种子页，因为“能动者”是这篇文章的收束点，也是从财富、学习、思维模型与行动策略之间穿过去的主体性概念。
- 这篇文章应先保留为 source guide，而不是直接扩展 [[wiki/topics/Wealth Creation]]。它的贡献不是财富公式本身，而是解释哪些旧观念会阻止人进入可复利、可反馈、可杠杆的行动结构。
- 文章中的重尾、乘法世界、可迁移技能和复利，应连接到既有的 [[wiki/concepts/Heavy-Tailed Distribution]]、[[wiki/concepts/Multiplicative World]] 与 [[wiki/topics/Judgment Under Leverage]]，不需要重复写成新的概念页。
- “稳态生存逻辑”可以暂时停留在 source 层。它像一个解释性框架，但还需要更多来源支撑，才能提升为独立概念或综合页。^[inferred]
- 文章与 [[wiki/topics/Mental Models]]、[[wiki/topics/Thinking in Systems]] 的连接在于：旧观念不是单点错误，而是一整套由资源、规则、风险和反馈共同塑造的心智模型。

## Open Questions
