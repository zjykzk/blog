# 光速不变与相对性原则问答 Source Guide

> Source: conversation:2026-05-26

## Capture Policy

这页保留一段围绕狭义相对论入门问题的中文教学追问。它保存的是问答链条中的解释层、类比、最小公式和概念边界，不把这段对话提升为完整物理学教材或严格历史论文。

## What It Covers

这段问答围绕四个连续问题展开：光速不变如何进入狭义相对论、为什么要提出这个假设、什么是相对性原则、什么是非惯性参考系。它的价值在于把相对论中容易被误解的几个入口概念连成一条学习路径：先区分“公设”和“推导”，再说明公设背后的理论和实验动机，最后把惯性系与非惯性系的边界讲清楚。

### 原始问题链

```text
1. 光速不变怎么推导？
2. 那为什么得出这个假设？
3. 什么是相对性原则？
4. 什么是非惯性参考系？
```

这四个问题形成一条连续教学线：第一个问题追问狭义相对论的数学结构，第二个问题追问公设的来源，第三个问题追问公设背后的原则，第四个问题补足“惯性系”这个前提的反面。

### 光速不变不是从普通速度叠加中推出的

光速不变在狭义相对论中是基本公设之一，不是从牛顿力学或伽利略速度叠加中推导出来的。真正被推导出来的是：如果所有惯性系中的物理规律相同，且真空光速对所有惯性观察者相同，那么时间、长度、同时性和速度叠加规则必须改变。

最短结构是：设两个惯性系 `S` 和 `S'`，`S'` 以速度 `v` 沿 `x` 方向相对 `S` 运动。光在 `S` 中满足：

```text
x^2 + y^2 + z^2 = c^2 t^2
```

如果光速在 `S'` 中也为 `c`，则应有：

```text
x'^2 + y'^2 + z'^2 = c^2 t'^2
```

为了让光锥形式在两个惯性系中都成立，坐标变换不能再是伽利略变换，而要变成洛伦兹变换：

```text
x' = gamma (x - vt)
t' = gamma (t - vx/c^2)
gamma = 1 / sqrt(1 - v^2/c^2)
```

这个变换保持时空间隔结构：

```text
x^2 - c^2 t^2 = x'^2 - c^2 t'^2
```

因此，如果某束光在一个参考系中满足 `x/t = c`，在另一个惯性系中仍会满足 `x'/t' = c`。

更准确的学习顺序是：不是用普通速度叠加推出光速不变，而是用光速不变逼迫普通速度叠加公式改写。相对论速度叠加公式是：

```text
u' = (u - v) / (1 - uv/c^2)
```

当 `u = c` 时：

```text
u' = (c - v) / (1 - v/c) = c
```

### 为什么会提出光速不变假设

光速不变不是凭空设定，而是由麦克斯韦电磁理论、以太实验失败和相对性原则之间的张力推动出来的。

第一，麦克斯韦方程推出真空电磁波速度为：

```text
c = 1 / sqrt(mu0 epsilon0)
```

这个速度由真空电磁常数给出，看起来不像普通物体速度那样需要额外说明“相对于谁”。在牛顿力学图景中，速度应依赖参考系；但电磁理论似乎给出一个固定传播速度。

第二，以太假说试图保留绝对参考系。若光速 `c` 是相对于静止以太的速度，地球在以太中运动时，沿地球运动方向和垂直方向测到的光速应有细微差异。迈克耳孙—莫雷实验没有测到这种预期差异，削弱了“以太风”解释。

第三，爱因斯坦选择保留相对性原则和麦克斯韦方程，而不是保留牛顿式绝对时间。如果所有实验都找不到特殊静止系，就不应假设它作为物理定律的隐藏背景。于是狭义相对论以两个原则出发：所有惯性系中的物理规律相同；真空光速对所有惯性观察者相同。

可压缩为：

```text
相对性原则 + 麦克斯韦方程 + 没有发现以太
=> 光速不变作为狭义相对论公设
```

这不是纯数学定理，而是一个被理论一致性和实验结果共同支持的物理原则。

### 相对性原则

相对性原则指：所有匀速直线运动的惯性参考系都是等价的，物理规律在它们里面应具有相同形式。它不是“所有观点都一样对”，也不是“什么都是相对的”，而是说物理定律不应偏袒某一个匀速参考系。

最直观的例子是匀速平稳行驶的高铁或船舱。在封闭车厢里把球竖直抛起，球仍会落回手里；只靠车厢内部的普通力学实验，无法判断车厢是静止还是匀速前进。只有加速、刹车、转弯时，车厢内部才会出现可察觉的特殊效应。

相对性原则包含三层意思：

1. 速度不是绝对属性，只能说一个物体相对于另一个物体多快。
2. 没有一个“宇宙标准静止系”天然更真实。
3. 物理规律的形式不能依赖观察者选择哪个惯性参考系。

在狭义相对论中，这个原则从力学扩展到全部物理规律，尤其包括电磁学。如果麦克斯韦方程在所有惯性系中形式相同，而它推出真空光速是 `c`，那么每个惯性系测到的真空光速都应是 `c`。

这也把客观性从“找到唯一真实视角”改写为“找到不同参考系之间保持不变的结构”。^[inferred]

### 非惯性参考系

非惯性参考系是自身在加速、减速、转弯或旋转的参考系。与它相对的是惯性参考系：在惯性系中，不受外力的物体保持静止或匀速直线运动。

刹车公交车是最简单的例子。车内乘客会感觉身体被一股力向前推；路边观察者则会说，不是有力把人向前推，而是人的身体由于惯性保持原来的向前运动，车突然减速了。车厢这个参考系自己在减速，所以车内描述需要引入一个“向前的惯性力”。

旋转木马也是非惯性系。坐在旋转木马上会感觉自己被向外甩，这种感觉在旋转系内可描述为离心力；站在外部近似惯性系中的观察者则会说，身体需要不断改变运动方向，真正需要解释的是向心力如何提供曲线运动。

非惯性系的关键不是“不能使用”，而是“参考系本身不安分”。若仍想在这种参考系中保持类似牛顿第二定律的形式，就必须加入由参考系运动造成的有效力，例如平动惯性力、离心力、科里奥利力。

最短区分是：

```text
惯性系：异常运动通常来自真实受力
非惯性系：异常运动可能来自参考系自身加速或旋转
```
