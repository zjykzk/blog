# Simon Sciences of the Artificial Source Guide

This page preserves the design-theory anchor from Herbert A. Simon's *The Sciences of the Artificial*.

## Source Claim

The MIT Press catalog describes Simon's book as proposing a science of design and treating artificial systems as a valid field of study.

Secondary design sources repeatedly cite Simon's definition that design devises courses of action aimed at changing existing situations into preferred ones.

## Use For Application Versus Creation

This source clarifies why design is a bridge between [[wiki/concepts/Application]] and [[wiki/concepts/Creation]]. It begins with an existing situation and a preferred one, so it must fit action to constraints. But it can also invent a new state of affairs, so it can become creative work.

In the local distinction, Simon helps locate the hinge: application changes action within a situation; creation changes the possible order of the situation. ^[inferred]

## Limitations

The exact design definition is cited here through secondary accessible pages as well as the MIT Press catalog record. Verify against the book text before treating wording-level claims as primary.
