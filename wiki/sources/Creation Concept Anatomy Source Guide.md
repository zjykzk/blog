# Creation Concept Anatomy Source Guide

This page preserves the knowledge extracted from the `ljg-learn` concept-anatomy note on [[wiki/concepts/Creation]].

## Source Claim

The source defines creation as bringing something not-yet-existing into existence, or reorganizing existing materials into a new form, meaning, or function.

It explicitly rejects the common reduction of creation to invention from nothing. In the source, creation is usually generated from materials, constraints, dissatisfaction, and the need for a new order.

## Useful Distinctions

- Creation versus pure novelty: not everything new is ordered enough to matter.
- Creation versus discovery: discovery reveals something already there; creation stabilizes a new possible form.
- Creation versus application: application fits an abstraction to a situation; creation can reshape the abstraction and the situation together. ^[inferred]
- Genius metaphor versus ecology metaphor: "genius burst" hides accumulation and conditions, while "ecology" highlights materials, audience, tradition, tools, and constraints. ^[inferred]
