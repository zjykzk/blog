# Mobu Notes Source Guide

`mobu/` 目前是个人知识与领域素材的大池子，适合作为 raw/domain input 层，而不是直接作为稳定 wiki 层。

## Current role

- 读书笔记
- 工作思考
- 育儿与学习相关材料
- 管理与工具类笔记

## Promotion rule

只有那些满足下面条件的笔记，才建议进一步提升到 `wiki/`：

- 脱离时间上下文后仍然成立
- 可以作为长期链接目标
- 值得持续维护
- 不只是摘录，而是已经形成稳定判断
