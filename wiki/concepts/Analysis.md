# Analysis

Analysis is the act of opening a whole along its own structure: distinguish its parts, relations, and constraints, then recombine them into an understanding that can guide action.

Its central danger is mistaking disassembly for understanding. A useful analysis is reversible: it can split a confused object into visible parts, but it can also return those parts to a living whole. ^[inferred]

## Core Structure

- Analysis starts from a whole that is too entangled to handle directly.
- It separates parts, relations, and constraints so the object gains visible structure.
- It works best when the cut follows the object's own grain rather than an arbitrary external grid.
- It matures only when the resulting structure changes what can be judged or done.

A compact formula is:

```text
Analysis = split by grain + map relations + recombine into action
```

## Analysis and Synthesis

The immediate counterpart of analysis is synthesis. Analysis opens the object; synthesis puts the object back together.

When analysis is isolated from synthesis, it can become sterile fragmentation. When synthesis is isolated from analysis, it can become vague impression. The stronger pattern is a reversible loop: split enough to see, recombine enough to act. ^[inferred]

## Relation to Thinking Tools

[[wiki/topics/Categorical Thinking]] is one concrete form of analysis: it selects a dimension and cuts a mixed object into classes.

[[wiki/topics/Thinking in Systems]] pushes against the failure mode of analysis by asking how the separated parts reconnect through feedback, delay, structure, and leverage points.

[[wiki/topics/Problem Framing]] often precedes analysis because a bad frame makes even careful decomposition answer the wrong question.

[[wiki/topics/Critical Thinking]] audits analysis by asking whether the cut, assumptions, evidence, and omitted alternatives are trustworthy.

## Failure Modes

- Cutting too coarsely leaves only slogans.
- Cutting too finely loses the living object.
- Treating every object as a static machine hides time, context, emergence, and participation. ^[inferred]
- Staying in analysis can become a way to avoid the risk of action. ^[inferred]

## Source Compression

The concept-anatomy source compresses analysis into one sentence: analysis is not breaking things apart, but turning confusion into a structure that can be grasped again.
