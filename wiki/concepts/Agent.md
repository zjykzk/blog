# Agent

Agent 的核心任务目前可以归纳为三件事：

- 知道大模型能干什么、不能干什么
- 管理上下文
- 调用工具

这一定义更偏执行系统视角，而不是把 agent 仅仅理解为“会聊天的模型”。
