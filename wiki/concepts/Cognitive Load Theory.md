# Cognitive Load Theory

Cognitive Load Theory explains learning as a process constrained by working-memory bandwidth: new information must pass through a narrow processing gate before it can become durable schemas in long-term memory.

## What It Is

The theory separates the learner's long-term storage problem from the learner's active processing problem. Long-term memory can hold a large number of schemas, but working memory can only coordinate a small number of interacting elements at once.

Learning fails when the material, explanation, environment, or tool setup overloads working memory before useful schemas form.

## Core Loads

- Intrinsic load is the complexity imposed by the learning material itself: how many elements must be coordinated together.
- Extraneous load is irrelevant burden created by presentation, environment, distraction, poor explanation, or tool friction.
- Germane load is often used for the useful effort of schema construction, although newer interpretations may fold it back into the management of intrinsic load. ^[ambiguous]

The practical teaching rule is to reduce extraneous load and sequence intrinsic load so the learner can build schemas instead of merely surviving information pressure.

## Schema Compression

A schema compresses many related elements into one usable cognitive unit. What looks like seven separate symbols to a novice can become one chunk to an expert.

This explains why prior knowledge changes apparent difficulty: the same material can be many working-memory elements for a novice and a few schema calls for an expert. ^[inferred]

## Instructional Implications

For novices, direct instruction, worked examples, explicit steps, and repeated near-transfer practice reduce search cost and protect scarce working-memory bandwidth.

For learners with stronger prior schemas, guidance can fade. More variation, transfer, and exploration become useful because the learner can absorb higher intrinsic load without being overwhelmed.

AI tutoring should follow the same constraint: it should reduce extraneous load and guide schema formation, not complete the learner's cognitive work as a doer.
