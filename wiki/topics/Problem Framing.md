# Problem Framing

一个问题可以先被看作：输出不符合预期。

这个定义虽然简单，但它迫使我们先把“预期”说清楚，而不是直接跳进解决方案。

## Core elements

当前笔记里提到的问题要素包括：

- 输入
- 输出
- 限制条件
- 角色
- 障碍
- 预期

## Problem-solving direction

解决问题时，至少要看三件事：

- 核心要素是什么
- 当前差异在哪里
- 应该选择什么策略
