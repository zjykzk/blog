# Modern Software Engineering

当前已有笔记里，现代软件工程最核心的一句定义种子是：

- 使用经验主义和科学的方法解决软件问题

这一定义值得保留，因为它把软件工程从“纯编程技巧”拉回到更可验证的方法论：

- 观察现实
- 建立假设
- 通过反馈修正
- 用经验与证据而不是纯直觉推动工程决策
