* 速读

{一张卡片，给两种人：还没决定要不要读全文的，和半年后回来只想要核心的。三句话，一句比一句细——下面那个长故事，就是把这三句一拍一拍展开。卡片最后写、最前放：没读懂全文，压不出来。}

** 一句话

{平实一句说完整篇：主角卡在什么困境、作者押了什么、结果怎样。不是上面 title 那句灵魂句（那句炼成警句、要 subtitle 兜底），是谁读一遍都知道"这论文干了什么"的话。}

** 大想法

{只能带走一个想法的话，是哪个。和「洞见」是同一颗核——这里先亮出来给方向，「洞见」在故事走完后落地、再给第二个用处。两处别用同一句话。}

** 只记三件事

{半年后忘掉大半，这三条忘不掉：困境为什么值得解、命题靠什么撑住、走出论文带走什么。「总目标」五件事压成三件，是重建故事的承重点——不是押注（那在「检验」），不是动手清单（那在「启发」）。}

* 问题

{搭台。一个具象示例——主角是谁、面前撞着什么——把读者拉到困境现场。一两句话说完。}

{三拍叙事，同一例子贯穿：(1) 亲历困境——主角 + 困境 (2) 旧路撞墙 + 立尺——前作在这例子上卡在什么两难、问题域水位在哪（base rate：现状到哪、典型一步多大）(3) 转折预告——本论文作者看到了别人没看到的入口，只引方向不展开机制，命题整句留给「翻译」开头。}

{末句给「翻译」节开口子——让读者已经隐约知道转折在哪里。}

{无明显前作时两难可省、尺照立（"之前没人敢碰"也是水位），直接「亲历→转折预告」。}

* 翻译

{推进。不重述问题。开头先亮命题——作者敢押的那句大胆猜想，亮到让人想反驳；整节就是看作者怎么把这句话撑起来。沿用「问题」里那个具象例子，在它上面一步步打开论文的方法。}

{命题亮出后立锚点——锚点是方法在那个例子上的样子（比如"在那个图灵奖问题上，方法像分诊台"）。}

{一步步揭开，让读者解锁迷雾。揭秘动词："这时候你看……""于是发生了……""那这一步怎么办呢——他们想到……"——前一段说的事打开后一段的口子。}

{覆盖（都在那个例子上）：命题是什么、解法怎么动（每个组件是命题底下哪根柱子）、结局长什么样（两三组数字挂回「问题」立的尺）、理解全文的钥匙概念。子标题按内容需要组织。}

{豁免：仅当机制细节同一例子撑不住时，引入"子例子"作为放大镜——保持父例子的问题域，不换地图。}

{末句给「核心概念」开口子——让读者已经感觉到有几件道具在动。}

* 核心概念

{故事道具。挑 3 个主角手里那几件不可少的东西——少了任一件，这故事就讲不通。逐个拆解，每个都回到「问题」那个例子上落地。}

{每个道具：一句话说清是什么、在那个例子上长什么样（少了它在例子上会怎样）、为什么少了它就跟不上后面的内容。读者每解锁一个，迷雾就散一片，对例子的理解再深一层。}

{末句给「洞见」开口子——读者已经准备好接故事的内核。}

* 洞见

{内核。一句话：这个故事真正留下的那个新视角——原来还能从这个角度看。说出角度，再补一句它除了这篇论文还能看什么。}

{脱离论文上下文，这句话本身还有没有力量？不要复述结论，要提炼听众走出门带走的那一句。没有就直说没有。}

* 博导审稿

{故事评审。像带了二十年研究生的博导在办公室跟学生聊——这个故事讲得真不真、值不值得认真对待。}

{选题眼光（困境真不真）、方法成熟度（找未被讨论的根本预设）、实验诚意（结局数字经不经得起追问）、声称的分量（放回领域水位上称——超常声称要超常证据）、写作功力。}

{判决：strong accept / weak accept / borderline / weak reject / strong reject，一句话理由。}

{末句把命题送出门——接「检验」。}

* 检验

{命题出门，走两步。}

{第一步生活测：挑一个读者熟到不用解释的场景，把命题放进去跑——哪儿站住了（命中）、哪儿失灵了（破绽）。破绽是命题的边界，不是污点。只用一个主场景。命题下不了生活的水就明说，不硬测。}

{第二步押未来：X 条件下该看到 Y。领域走向 / 现实行为 / 产品形态，至少押一注，写成能被打脸的样子。推不出预测就直说——这本身是个发现。}

{这节管"信不信"，「启发」管"用不用"。}

* 启发

{听众的故事。这个故事接到我的生活里——迁移 / 混搭 / 反转 三个视角试探，命中展开，全没命中说「没有」。落点在"你可以___"，不在"值得思考___"。}
