# 分类思维 source note

沉淀后的主题页：[[wiki/topics/Categorical Thinking]]

## 原始摘录

### 一句话

分类思维，是用一个有用的维度，把混乱切成可操作的差异。

### 核心定义

分类思维不是简单贴标签，而是：

1. 先找一个划分维度
2. 再按这个维度把对象分开
3. 让同类内部更像，异类之间更不同
4. 最后用这个划分去支持理解、决策或行动

一个好分类，价值不在整齐，而在有用。

### 常见分类维度

- 按时间分：短期 / 长期
- 按可控性分：可控 / 不可控
- 按确定性分：确定 / 不确定
- 按对象分：人 / 事 / 机制
- 按作用层级分：现象 / 结构 / 激励 / 目标

### 作用

1. 降噪：把混在一起的东西拆开
2. 提高判断质量：不同类的问题通常要用不同解法
3. 暴露隐藏变量：一旦开始分类，就会暴露自己真正采用的划分维度

### 误区

- 把分类当天然存在
- 分得太早
- 分类后不回看
- 只分不连

### 压缩句

- 不会分类的人，看到的是一团；会分类的人，看到的是结构。
- 分类是工具，不是现实本身。
