# 分类的方法 source note

沉淀后的主题页：[[wiki/topics/Methods of Classification]]

## 原始摘录

分类方法的差异，本质上是按什么维度切对象。

### 六种常见方法

1. 属性分类：按对象具有什么特征来分
2. 功能分类：按对象拿来做什么来分
3. 结构层级分类：按对象处在哪一层来分
4. 因果角色分类：按对象在因果链或因果环中扮演什么作用来分
5. 时间分类：按对象处在什么阶段或时间位置来分
6. 可操作性分类：按我能不能动它、怎么动它来分

### 更深一层的方法

- 原型分类：按像不像典型成员来分
- 家族相似分类：按局部相似网络来分，而不是找一个共同本质
- 二分法 / 多分法：按切分类数来分
- 分类树 / 分类矩阵：按多维组合来分

### 使用原则

- 先问分类是为了什么
- 再问真正关心的差异是什么
- 最后问分完之后能不能支持不同动作
