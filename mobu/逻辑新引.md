# 逻辑新引
- 推论
    - 外部推论：以未经解析的语句作元素的推论。
        - 选取推论
            - 相容
                - 不相容：肯定其一否定另外一个
                - 相容：肯定其一不能否定另外一个
            - 穷尽
                - 穷尽：否定其一，肯定另外一个
                - 不穷尽：肯定其一，不能否定另外一个
        - 条件推论
            - 如果p，那么q
            - p是q的充分条件，p真则q真
            - q是p的必要条件，q假则p假
            - p是q的充分必要条件，p真则q真，p假则q假，q假则p假，q真则p真
        - 二难式
            - 逻辑观点看，基于条件推论和选取推论，是多余的
            - 形式一：简单肯定前置条件，如果甲则丙，如果乙则丙;甲或乙->丙
            - 形式二：简单否定后件，如果甲则乙，如果甲则丙;非乙非丙->非甲
            - 形式三：复合肯定前置条件，如果甲则乙，如果丙则丁;甲或丙->乙或丁
            - 形式四：复合肯定后置条件，如果甲则乙，如果丙则丁;非乙非丁->非甲非丙
            - 破局：
                - 否认前置条件为后置条件的充分条件
                - 否认选取语句是互不相容或者是共同穷尽的
                - 做一个二难式使结论与之相反
    - 内部推论：以经解析的语句作骨架的推论，并不是每一语句可能表示的特殊内容，而是语句内部的逻辑结构。
        - 语句：形式明显而且语法完备。逻辑只研究直叙语句（proposition，命题），比如：地球是圆的。并 **只研究** 语法结构与语义条件， **不研究** 语句的特殊内容与文法结构。
        - 传统逻辑中主宾词式分类：
            - 1. 全肯定语句（universal affirmative sentence）=》A =》一切S是P
            - 2. 全谓否定语句（universal negative sentence）=》E =》没有S是P
            - 3. 偏谓肯定语句（particular affirmative sentence）=》I =》有些S是P
            - 4. 偏谓否定语句（particular negative sentence）=》O =》有些S不是P
            - 说明
                - 肯定和否定是形式的 **性质** ，全谓和偏谓是形式的 **分量** 。
                - 普及（distributed&undistributed），S自身的全谓决定了是普及，偏谓不普及；P要看自己是否和S的关系是全部还是部分，来决定是否普及。
