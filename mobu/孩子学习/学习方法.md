# 学习方法
- 专注校内学习，不是校外
- 专注课堂，不是提前学
- 专注课本，不是教辅
- 专注老师布置的作业，不是妈妈牌作业
- 专注理解，不是死记硬背
- 自己花功夫，不依赖app讲解
- 全学科阅读，培养强大的阅读能力
- 锻炼自学能力，自己预习自己复习
- 各科
    - 数学
        - 做例题，自己推导一遍，全部做一遍
            - 题目是例题变形和组合
            - 读题能力
        - 自己推演公理，定理
    - 语文
        - 基础
            - 朗读课文，朗朗上口，非常流畅
            - 生字词和组词
            - 注解搞清楚
            - 每个字词都要读懂，彻底理解
            - 文学常识
                - 日常大量阅读
        - 阅读理解
            - 自主阅读
            - 套路，很重要：揣摩出题人思路
                - 假设老师想要你怎么答
            - 做错了，琢磨怎么错的，和老师的答案差哪里
            - 注意：会有自己的理解，不一定要和老师的一样，但是得按考试规则
        - 作文
            - 技术活
            - 主题书单
            - 会读一定会写作文，言之有物，词汇
            - 别帮倒忙
                - 给孩子很多作业
            - 遇到问题的时候帮忙
            - 套路，很重要：审题，揣摩出题人思路
        - 古诗词
            - 不花背很多内容，按照老师要求就可以，不要家长自己加
            - 容易背的原因
                - 韵律
                - 内容理解
            - 充分利用晨读时间，培养语感，节奏
    - 英语
        - 基础
            - 课内单词
            - 单词发音（听）
            - 基本语法（读）
            - 构词法（初中，高中）
    - 政治
        - 抄书，抄关键词，有自己的理解，不能用自己的语言，必须要课本中的词
    - 历史
        - 阅读多，感觉轻松，亲切
        - 归纳表，可以区分不同
        - 工具书，看字典
        - 历史地图，印象比较深刻
        - 核心关键字和政治一样
        - 历史博物馆，遗迹==
    - 地理
        - 画图，各种图
